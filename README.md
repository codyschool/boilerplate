# CodySchool Boilerplate

> Boilerplate component using Vite, LitElement and TypeScript

## Usage

```bash
$ npm run dev
```

You can view the development server at `http://localhost:3000`

## Production

```bash
$ npm run build
```

You can view the deploy by creating a server in `dist`

```bash
$ cd dist && npx serve
```
